# TO-DO LIST MAKER

It is a CLI based to-do list application, programmed using python, that allows users to add, view, complete, edit, and delete tasks efficiently.

## Features
- Add new tasks  
- View existing tasks  
- Mark tasks complete  
- Edit tasks  
- Delete tasks  
- Exit the program

## To Use
- Clone the repository
- Go to the local folder
- Run the program
